Primary call-to-action button — use for any decisive action (agendar diagnóstico, enviar, falar com a equipe).

```jsx
<Button variant="primary" size="lg">Agendar diagnóstico</Button>
<Button variant="metal">Conheça os cases</Button>
<Button variant="outline" surface="dark">Áreas de atuação</Button>
```

Variants:
- `primary` — solid emerald, ivory label. The default decisive action.
- `metal` — platinum-sheen fill with dark label. Reserve for one premium moment per view.
- `outline` — hairline border; pass `surface="dark"` when placed on an emerald section.
- `ghost` — text-only, for tertiary actions.

Labels are UPPERCASE Jost, tracked. Keep them short and verb-first. Near-square corners (4px) — never pill.
